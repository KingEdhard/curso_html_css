Para mostrar imágenes al usuario haríamos uso de la etiqueta img. Esta etiqueta no requiere el uso de la etiqueta de cierre.
Aquí tenemos dos alternativas: la primera sería mediante nuestra ubicación local o mediante el uso de un link externo.

Otra forma de mostrar texto mediante local, sería directamente ordenando la ruta desde donde tenemos las imagenes guardadas.
Es recomendable durante el desarrollo de nuestros proyectos mantener una carpeta en nuestro directorio así: img

Para ajustar el tamaño de la imagen dentro de la etiqueta img, le indicaremos con la propiedad width el ancho en pixeles que queremos asignarle a la imagen. También el width manipula el height (alto ) de la imagen de manera proporcional al ancho, por ello, es recomendable no manipular el height, salvo casos especiales que lo ameriten.

Uso de la etiqueta form para crear formularios

El uso de la etiqueta form nos permite recopilar información del usuario. Dentro del cuerpo de esta etiqueta debemos iniciar con la etiqueta input, esta, al igual que la etiqueta img, puede ser cerrada al momento sin ningún problema. para que estas etiequetas tengan sentido, haremos uso de "label"
